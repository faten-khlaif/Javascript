# Javascript

Welcome to my JavaScript repository! This project serves as a practical showcase of the foundational skills, programming concepts, and interactive applications built while mastering modern front-end engineering.

## 🚀 Project Overview

This curriculum was designed to bridge the gap between architectural theory and practical execution. By building responsive interfaces and logic-driven applications from scratch, I developed a deeper understanding of how the core pillars of the web intersect to create modern user experiences.

### 🛠️ Core Technologies Used
* **HTML5:** Semantic structuring of web content and data.
* **CSS3:** Responsive layout design, visual styling, and custom animations.
* **JavaScript (ES6+):** Object-oriented logic, DOM manipulation, state management, and asynchronous API integration.

## 🎮 Featured Applications Built

Inside this portfolio, you will find the following completed projects:

1. **Interactive Portfolio Website:** A personalized hub displaying my development timeline, coding skills, and hosted projects.
2. **Dynamic Calculator:** A functional, grid-aligned arithmetic engine designed with native JavaScript mathematical logic.
